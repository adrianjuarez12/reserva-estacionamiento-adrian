# reserva-estacionamiento-adrian
Sistema automático de reserva de estacionamiento para Prima AFP

Este codigo reserva el estacionamiento para Adrian Juarez de lunes a jueves a las 00 horas de forma automática, sin necesidad de que tenga que realizarlo manualmente levantandose a la medianoche cada día compitiendo entre 400 trabajadores por los 25 puestos de estacionamiento disponibles diariamente.
